## 1. 软件概述

厘清记账软件是一款面向个人用户的极简记账应用，支持 Android 和 iOS 双平台。

**核心功能**：
- 日常收支记账管理
- AI 智能语音记账
- 月度账单浏览与搜索
- 消费趋势统计与分类排行
- Excel/CSV 数据导出
- 深色模式 / 浅色模式

**技术架构**：
- 移动端：Flutter 跨平台框架
- 服务端：Node.js (Fastify) + Prisma ORM + SQLite

## 3. 安装与登录

### 3.1 安装应用

[截图占位：应用安装界面 / 应用图标]

从应用商店搜索「厘清」下载安装，或通过提供的 APK / IPA 安装包直接安装。

### 3.2 邮箱验证码登录

当前版本采用邮箱验证码登录方式，首次登录成功后自动创建账号，无需单独注册页面。

[截图占位：登录页 - 输入邮箱]

1. 输入邮箱地址
2. 点击「发送验证码」按钮
3. 查收邮件中的 6 位数字验证码
4. 输入验证码完成登录
5. 首次登录自动注册账号

[截图占位：验证码输入页面]

### 3.3 用户协议与隐私政策

[截图占位：登录页底部的协议勾选框]

登录前请确认已阅读并同意《用户协议》和《隐私政策》，可在登录页点击对应链接查看详情。

## 4. 首页概览

[截图占位：首页完整截图]

首页是应用的核心面板，展示以下信息：

- 交易分类图标和名称
- 交易标题
- 交易金额（绿色为收入，红色为支出）
- 交易时间

### 4.3 快捷记账

[截图占位：首页的快速记账入口]

点击首页右下角的「+」按钮快速添加一笔交易。

## 5. 记账管理

### 5.1 新增交易

[截图占位：新增交易弹窗/页面]

填写以下信息创建一笔交易：

| 字段 | 说明 | 必填 |
|------|------|------|
| 标题 | 交易名称 | 是 |
| 金额 | 收入或支出金额 | 是 |
| 方向 | 收入 / 支出 | 是 |
| 分类 | 选择对应的类别 | 是 |
| 日期 | 交易发生时间 | 是 |
| 备注 | 补充说明 | 否 |
| 商家 | 交易商户名称 | 否 |

### 5.2 编辑交易

[截图占位：编辑交易页面]

在账单列表或首页最近交易中点击任意交易记录，即可在弹窗中修改所有字段。

### 5.3 删除交易

[截图占位：删除确认弹窗]

在编辑交易弹窗中点击删除按钮，确认后即可删除该笔交易。

### 5.4 分类管理

**预设分类**：应用预置了常用的收支分类：

| 支出分类 | 收入分类 |
|---------|---------|
| 用餐、公交、购物、生活、水电 | 工资、兼职、理财 |
| 加油、旅行、就医 | 转账、报销 |

[截图占位：分类管理页面]

用户可在「个人设置」中：
- 新增自定义分类
- 编辑分类名称、图标
- 删除未使用的分类
- 拖拽排序

## 6. 账单浏览

[截图占位：账单页完整截图]

### 6.1 月度账单

账单页面以「天」为单位按月分组展示所有交易记录：
- 每天显示收入和支出的小计金额
- 每条记录显示分类图标、名称、金额

### 6.2 筛选与搜索

[截图占位：筛选和搜索功能]

- **收支类型筛选**：全部 / 仅支出 / 仅收入
- **月份切换**：通过日历选择查看任意月份
- **关键词搜索**：支持按标题、备注、分类名称搜索

### 6.3 分页加载

[截图占位：下拉加载更多]

账单列表采用游标分页（Cursor-based Pagination）设计，向下滚动自动加载更多交易记录，性能流畅。

## 8. AI 智能记账

[截图占位：AI 记账聊天界面]

### 8.1 自然语言记账

通过对话的方式记录交易，例如输入：
- "中午吃饭花了 35 块"
- "今天发了工资 8000"
- "买了杯咖啡 28"

AI 会自动识别金额、方向、分类等信息，并给出确认提示。

### 8.2 语音输入

[截图占位：语音输入界面]

支持语音识别输入，长按麦克风按钮说出交易内容，自动转为文字后提交 AI 处理。

### 8.3 AI 配置

[截图占位：AI 设置页面]

在「个人设置 - AI 设置」中可配置：
- AI 服务地址（支持 OpenAI 兼容接口）
- API Key
- 模型名称
- 自定义提示词

## 9. 数据导出

[截图占位：导出页面]

### 9.1 CSV 导出

导出为逗号分隔值文件，可用 Excel、WPS 等软件打开。包含 BOM 头，中文显示正常。

- 表头绿色背景白色文字
- 支出标记为红色字体

- 冻结首行，方便滚动浏览

### 9.3 导出范围

[截图占位：导出范围选择]

选择起止月份导出指定时间范围的交易数据。

## 11. 常见问题

**Q：如何找回密码？**
A：使用邮箱验证码登录功能，输入邮箱获取验证码即可登录，无需密码。

**Q：数据会丢失吗？**
A：数据存储在云端服务器，更换设备登录同一账号即可恢复数据。

**Q：支持多设备同步吗？**
A：支持，登录同一账号即可在不同设备间同步数据。

**Q：如何注销账号？**
A：进入“个人设置”，点击“注销账号”并确认后，即可删除账户及相关个人数据。
