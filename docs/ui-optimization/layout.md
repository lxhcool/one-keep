# 布局结构优化建议

## 优化原则

1. **呼吸感**：增加适当的留白，避免信息过于密集
2. **视觉层次**：通过间距和大小建立清晰的信息层级
3. **一致性**：统一间距规范，提升整体协调性
4. **易用性**：优化触摸目标大小，提升操作体验

## 间距系统优化

### 当前问题
- 间距值过多：4, 8, 10, 12, 14, 16, 18, 20, 24, 28, 32
- 缺乏统一规范，导致视觉不协调

### 优化方案：8pt 网格系统

```dart
class AppSpacing {
  // 基础单位：8pt
  static const unit = 8.0;
  
  // 标准间距
  static const xxs = 4.0;   // 0.5x - 极小间距
  static const xs = 8.0;    // 1x - 最小间距
  static const sm = 12.0;   // 1.5x - 小间距
  static const md = 16.0;   // 2x - 中等间距
  static const lg = 24.0;   // 3x - 大间距
  static const xl = 32.0;   // 4x - 超大间距
  static const xxl = 40.0;  // 5x - 极大间距
  static const xxxl = 48.0; // 6x - 巨大间距
  
  // 页面边距
  static const pagePadding = 20.0;  // 统一页面左右边距
  static const pageTop = 16.0;      // 页面顶部间距
  static const pageBottom = 110.0;  // 页面底部间距（导航栏）
  
  // 卡片间距
  static const cardGap = 16.0;      // 卡片之间的间距
  static const cardPadding = 20.0;  // 卡片内边距
  
  // 列表间距
  static const listItemGap = 12.0;  // 列表项之间的间距
  static const listSectionGap = 24.0; // 列表分组之间的间距
}
```

## 圆角系统优化

### 当前问题
- 圆角值混乱：8, 10, 12, 14, 18, 20, 24, 28
- 缺乏统一的视觉语言

### 优化方案：统一圆角规范

```dart
class AppRadius {
  // 基础圆角
  static const xs = 8.0;    // 小组件（chip、tag）
  static const sm = 12.0;   // 按钮、输入框
  static const md = 16.0;   // 小卡片
  static const lg = 20.0;   // 标准卡片
  static const xl = 24.0;   // 大卡片
  static const xxl = 28.0;  // 底部弹窗
  
  // 特殊圆角
  static const round = 999.0; // 完全圆形
  static const pill = 100.0;  // 胶囊形状
}
```

## 页面布局优化

### 1. 首页（HomePage）

#### 当前布局问题
- 余额卡片过大，占据过多空间
- 收入/支出卡片间距过小（14px）
- 最近记账标题样式不够突出

#### 优化方案

```
┌─────────────────────────────────────┐
│ [头像] 早上好，用户名    [通知]      │ ← 顶部 16px
│                                     │
│ ┌─────────────────────────────────┐ │
│ │     本月结余                [👁]  │ │
│ │     ¥ 12,345.67                 │ │ ← 余额卡片高度减少
│ └─────────────────────────────────┘ │   padding: 24px（原 28px）
│                                     │   间距 20px
│ ┌───────────────┐ ┌───────────────┐ │
│ │ [↑] 本月支出   │ │ [↓] 本月收入   │ │
│ │ ¥ 8,234.56    │ │ ¥ 20,580.23   │ │ ← 间距改为 16px
│ └───────────────┘ └───────────────┘ │
│                                     │   间距 24px
│ ━━ 最近记账              查看全部 → │ ← 使用装饰线
│                                     │   间距 16px
│ ┌─────────────────────────────────┐ │
│ │ [图标] 午餐          -¥45.00     │ │
│ │        餐饮 · 12:30              │ │
│ └─────────────────────────────────┘ │
│                                     │   间距 12px
│ ┌─────────────────────────────────┐ │
│ │ [图标] 工资          +¥8,500.00  │ │
│ │        收入 · 10:00              │ │
│ └─────────────────────────────────┘ │
└─────────────────────────────────────┘
```

**具体调整**：
- 页面左右边距：24px → 20px
- 余额卡片内边距：28px → 24px
- 余额卡片圆角：28px → 24px
- 收入/支出卡片间距：14px → 16px
- 卡片之间间距：统一为 16px 或 24px
- 列表项间距：10px → 12px

## 组件尺寸优化

### 触摸目标尺寸

根据 iOS 和 Android 设计规范，最小触摸目标应为 44x44pt：

```dart
class AppSize {
  // 最小触摸目标
  static const minTouchTarget = 44.0;
  
  // 图标尺寸
  static const iconXs = 16.0;
  static const iconSm = 20.0;
  static const iconMd = 24.0;
  static const iconLg = 28.0;

```

## 响应式布局建议

### 小屏幕（< 375px）
- 减少页面左右边距：20px → 16px
- 减少卡片内边距：20px → 16px
- 调整字体大小：适当缩小 10%

### 大屏幕（> 428px）
- 限制最大内容宽度：480px
- 居中显示内容
- 保持边距比例

### 平板（> 768px）
- 使用两栏布局
- 增加卡片最大宽度
- 优化横向空间利用
