✓ 使用装饰线增强标题视觉效果
✓ 列表项间距统一为 12px

## 余额卡片样式对比

#### 当前样式
```
背景：纯黑渐变 (#18181B → #09090B)
效果：沉闷、缺乏层次
```

#### 优化后样式

#### 当前样式
```
背景：深灰渐变 (#27272A → #18181B)
```

```
阴影：带色彩的阴影 rgba(5,150,105,0.3)
```


### 应用示例
```
页面顶部间距：16px (md)
卡片之间间距：20px (lg)
列表项间距：12px (sm)
卡片内边距：20px 或 24px
文字行间距：4px (xxs) 或 8px (xs)
```
