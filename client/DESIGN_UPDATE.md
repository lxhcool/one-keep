# 首页设计更新说明

## 更新内容

根据 Pencil 设计稿，对记账 app 首页进行了全面的视觉升级。

### 主要变更

#### 1. 顶部黄色区域

- **背景色**: 从 `#E9D35E` 更新为 `#FCD34D`（更明亮的黄色）
- **布局优化**:
  - 添加了状态栏显示（时间和电量）
  - 顶部显示头像和"鲨鱼记账"标题
  - 日期选择器采用新的布局（年份 + 月份 + 下拉图标）
  - 收入和支出并排显示，样式更清晰
- **字体调整**:
  - 标题字体大小: 20px, 粗体
  - 月份字体大小: 24px, 粗体
  - 金额字体大小: 28px, 粗体
  - 标签字体大小: 13px

#### 2. 交易记录列表

- **背景色**: 统一使用白色背景
- **日期头部**:
  - 背景色改为白色
  - 字体颜色: `#71717A`（灰色）
  - 字体大小: 13px
  - 内边距调整为 20px 水平，12px 垂直
- **记录项**:
  - 图标颜色改为黄色 `#FCD34D`
  - 图标大小: 24px
  - 文字颜色: `#18181B`（深色）
  - 字体大小: 15px
  - 内边距: 20px 水平，12px 垂直

#### 3. 底部导航栏

- **整体样式**:
  - 添加顶部边框（1px, `#E5E5E5`）
  - 高度: 80px
  - 使用 SafeArea 确保在刘海屏上正确显示
- **导航图标**:
  - 明细: `Icons.list`
  - 图表: `Icons.trending_up`
  - 发现: `Icons.explore`
  - 我的: `Icons.person`
  - 图标大小: 20px
  - 文字大小: 11px
- **记账按钮**:
  - 尺寸: 56x56px
  - 背景色: `#FCD34D`（黄色）
  - 圆形设计
  - 加号图标大小: 32px
- **颜色方案**:
  - 选中状态: `#18181B`（深色）
  - 未选中状态: `#71717A`（灰色）

#### 4. 整体页面

- **背景色**: 从 `#F4F4F4` 改为纯白色
- **移除了 SafeArea**: 让黄色区域延伸到状态栏

### 设计原则

1. **简洁明快**: 使用明亮的黄色作为主色调，营造积极的氛围
2. **清晰层次**: 通过颜色、字体大小和间距建立清晰的视觉层次
3. **一致性**: 统一的颜色方案和间距系统
4. **可读性**: 优化字体大小和颜色对比度，提升可读性

### 技术细节

- 所有颜色值使用十六进制格式
- 使用 Flutter Material Icons
- 保持了原有的功能逻辑不变
- 响应式设计，适配不同屏幕尺寸

### 文件变更

- `client/lib/features/home/presentation/home_page.dart`: 完整重构了 UI 组件
