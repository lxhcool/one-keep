# 各应用商店注册指南

## 1. 华为应用市场

**网址**：https://developer.huawei.com/consumer/cn/

**费用**：个人 0 元 / 企业 0 元

**要求**：
- 实名认证（身份证 + 人脸识别）
- 软件著作权证书
- 隐私政策链接

**上架流程**：
1. 注册华为开发者账号
2. 创建应用，填写应用信息
3. 上传 APK + 截图
4. 提交审核（约 1-3 个工作日）

## 2. 小米应用商店

**网址**：https://dev.mi.com/

**费用**：个人 0 元

**要求**：
- 实名认证
- 软件著作权证书
- 隐私政策

## 3. OPPO 应用商店

**网址**：https://open.oppomobile.com/

**费用**：个人 0 元

**要求**：
- 实名认证
- 软著或授权证明

## 4. Vivo 应用商店

**网址**：https://dev.vivo.com.cn/

**费用**：个人 0 元

**要求**：
- 实名认证
- 软著或授权证明

## 5. App Store（iOS）

**网址**：https://developer.apple.com/

**费用**：$99/年

**流程**：
1. 注册 Apple Developer Program
2. 在 App Store Connect 创建应用
3. 配置证书和描述文件
4. 上传 IPA（需 macOS + Xcode）
5. 提交审核（约 1-7 天）

## 注意事项

- 各商店的应用截图尺寸要求不同，建议准备至少 2 套尺寸
- 应用简介和关键词要针对每个商店独立优化
- 软著是大多数国内商店的硬性要求，建议优先办理
